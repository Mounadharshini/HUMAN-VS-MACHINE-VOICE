# Human Voice vs Machine Voice Classification System

## Objective
This project builds an AI system that classifies audio input as either:  
- **Human Voice**  
- **Machine Voice** (AI-generated / synthetic speech)  

The system helps detect whether a voice is natural or AI-generated using deep learning.

---

## Technologies Used
- **Python**  
- **Flask** – for the web interface  
- **TensorFlow / CNN** – for audio classification  
- **Librosa** – for audio feature extraction  
- **HTML / CSS / JavaScript** – for frontend animations and interactivity  

---

## Features
- Upload or record audio directly in the browser  
- Analyze audio and classify it as Human or Machine voice  
- Shows confidence scores for predictions  
- Dynamic and interactive UI with waveform visualization  

---

## Dataset
- Any dataset containing human and synthetic voices  
- Audio format: `.wav` or `.mp3`  
- Sample rate: 16kHz or 44.1kHz  
- Recommended: 1000 samples per class
